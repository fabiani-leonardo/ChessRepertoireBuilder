# ChessRepertoireBuilder ♟️

Un'applicazione in Python/Pygame per costruire e allenare il tuo repertorio scacchistico personale.

## Installazione
1. Clona questo repository.
2. Crea un ambiente virtuale e installa i requisiti: `pip install pygame-ce chess`
3. **ATTENZIONE:** Questo programma richiede Stockfish per funzionare.
   - Scarica Stockfish AVX2 da [stockfishchess.org](https://stockfishchess.org/download/)
   - Estrai il file `.exe` e mettilo nella cartella principale del progetto.
   - Assicurati che il nome nel file `gui.py` corrisponda al file scaricato.
4. Avvia il programma con `python gui.py`
